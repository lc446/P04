Tools and How to Play!

Play Quake

Playing Quake is most easily done by buying it on Steam. It just received an anniversary update bringing Quake's official launcher the easiest way to play.

To avoid having to make a Bethesda account to play online however, I highly recommend using the Quakespasm source port. All you need from your purchased copy of Quake is the "id1" folder, with its PAK1.pak and PAK0.pak, move that folder into the directory with Quakespasm and you are playing Quake!

Download Quakespasm here

Mapping Tools

So I've inspired you to make a map of your own. No fear! The main things you will need is a map editor and compilation tools.

First thing you need is the editor. The best is TrenchBroom. You will need to link TrenchBroom to a port of Quake in order to test out your map.

Link to latest TrenchBroom release on Github

The other thing you will need is compilation tools. These require some setup and I recommend this video from dumptruck_ds on YouTube as a means to get you fully started.

Link to latest ericw-buildtools release on Github
